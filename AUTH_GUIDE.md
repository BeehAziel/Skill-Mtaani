📝 HOW TO USE:

1. REGISTER NEW ACCOUNT
   • Click 👤 in top right
   • Click "Create one" link
   • Fill in all fields:
     - Full Name (required)
     - Email (must be unique)
     - Username (must be unique)
     - Phone (WhatsApp number)
     - Ward/Location
     - Password (min 8 characters)
     - Confirm password
   • Check terms & conditions
   • Click "Create Account"
   • Auto-login & redirected to platform

2. LOGIN TO EXISTING ACCOUNT
   • Click 👤 in top right
   • Enter email or username
   • Enter password
   • Optional: Check "Remember me"
   • Click "Sign In"
   • User button shows your initial
   • Full name displayed on hover

3. VIEW YOUR PROFILE
   • Click 👤 button
   • Click "My Profile"
   • See your info:
     - Avatar with initial
     - Full name & username
     - Email & phone
     - Member since date
     - Number of skills posted
   • View all your posted skills
   • Edit or delete skills

4. MANAGE YOUR SKILLS
   • Click 👤 button
   • Click "My Skills"
   • See all skills you've posted
   • Options to:
     - Edit skill details
     - Delete skill
     - View skill stats

5. POST A NEW SKILL
   • Click "+ Post Skill"
   • Fill in skill details
   • Your name auto-fills
   • Your phone auto-fills
   • Click "Post My Skill 🚀"
   • Skill appears immediately
   • Community post created
   • Toast notification shown

6. LOGOUT
   • Click 👤 button
   • Click "Logout"
   • User button resets to 👤
   • Redirected to home
   • Data preserved for next login

❓ TROUBLESHOOTING:

Data Not Saving?
  → Check if localStorage is enabled
  → Clear browser cache
  → Try non-incognito/private mode

Can't Login?
  → Check email/username spelling
  → Verify password is correct
  → Make sure Caps Lock is off
  → Try demo account

Can't Register?
  → Email already registered? Use different email
  → Username already taken? Choose different username
  → Password must be 8+ characters
  → Agree to terms & conditions
  → Valid phone format required

Profile Not Showing?
  → Make sure you're logged in
  → Refresh page
  → Check browser console for errors
